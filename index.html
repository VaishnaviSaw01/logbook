Project Overview

Why LogBook Exists

Many small businesses still rely on manual bookkeeping and fragmented recordkeeping. LogBook solves this by providing a cloud-hosted ledger system that centralizes business transactions and improves accessibility, reliability, and data protection.

Business Focus

Designed for customer records, supplier management, invoices, purchases, and payment tracking.

Deployment Process

GitHub-Based CI/CD

1. Push Code to GitHub

The source code is stored in a GitHub repository.

2. Connect Deployment Center

Azure App Service is linked with the repository.

3. Generate GitHub Actions

Azure creates the workflow for automatic builds and deployment.

4. Deploy Live

Every push triggers deployment to the Azure hosted application.
